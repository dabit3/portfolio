Building a real app.

Mobile app fetching stories from Freshly Presses - https://wordpress.com/fresh/


Get Started


$ ionic start freshlypressed // create project
$ cd freshlypressed
$ cd www
$ rm -rf index.html // delete index.html file
$ rm -f js templates // delete js and templates directory
$ touch index.html // create new index.html file
$ mkdir js // create new js directory
$ touch js/app.js // create new app.js file

What we did

Created new ionic project and cleaned out some files
